# 标注规范

这份文档用于 X-AnyLabeling 标注四川麻将图像。目标是统一口径，减少后续训练和评测时的歧义。

## 1. 框的边界规则

统一原则：**框尽量贴着牌的可见边缘，只包牌体，不额外吞进大块背景。**

- 只要牌面可见，就要标框，不要因为只露出一部分就跳过
- 牌被遮挡超过 70% 仍然要标，只要还能确认它是一张牌
- 叠牌、搭牌、压牌时，按每张牌单独框
- 画面边缘只露出半张牌，也照样标
- 框尽量贴着可见牌体边缘，不要把大块背景一起包进去
- 如果牌的一部分被手、桌边或其他牌挡住，框只包当前看得见的牌体区域，不要脑补被挡住的部分

## 2. 类别总览

分类一共 29 类：

- `w1` 到 `w9`：一万到九万
- `t1` 到 `t9`：一条到九条
- `b1` 到 `b9`：一筒到九筒
- `back`：牌背
- `unknown`：侧面、严重遮挡、不可读、非万条筒的牌

**字牌和花牌出现时，分类一律标 `unknown`。**

## 3. 易混淆类别判别要点

### 3.1 `4条` vs `5条`

- 先看内部条纹的总数，再看排布是否对称
- `4条` 通常更容易形成较规则的四组分布，`5条` 会多出一个额外条纹或中心结构
- 画面旋转时不要只靠外轮廓猜，优先看中间区域有没有“多一根”的信息
- 如果只看到一半且条纹数量无法数清，直接标 `unknown`

### 3.2 `6条` vs `9条`

- 这组最容易被旋转方向误导，先恢复牌面朝向再判断
- `6条` 和 `9条` 的差异通常体现在上下结构方向，不要只按“像不像”判断
- 若图案方向看不清，或者画面模糊到无法确定上下关系，宁可标 `unknown`

### 3.3 `2条` vs `3条`

- 重点看条纹根数，不要被透视压缩骗到
- `2条` 通常是两组清晰分开的元素，`3条` 会多一组结构
- 如果只看到局部、怀疑有一根被遮挡，不要补脑，无法确认就标 `unknown`

### 3.4 各万字牌区分

- 万字牌主要靠数字字形和“万”字整体结构区分
- 先确认牌面是否正向，再看横竖笔画、转折、缺口和相对位置
- 不要只看“这一团字差不多”，尤其是中低清晰度下更要逐笔画比
- 如果数字部分被遮挡、糊掉或反光严重，无法稳定区分时标 `unknown`

### 3.5 `2筒` vs `3筒`

- 先数圆筒数量，再看空间排布
- `2筒` 往往是两枚主体，`3筒` 会形成三枚排布关系
- 透视、反光、遮挡很容易让一枚筒子“消失”，遇到这种情况不要硬判
- 只要不能稳定确认是 2 个还是 3 个，就标 `unknown`

## 4. `back` 与 `unknown` 的使用边界

### `back` 的典型场景（至少记住这 3 类）

- 只看到完整或大部分牌背图案，能明确不是牌面
- 对手手牌或牌墙朝向镜头的一排牌背
- 背面有轻微反光，但仍能确认是统一牌背纹理

### `unknown` 的典型场景（至少记住这 3 类）

- 牌侧面朝向镜头，只能看到厚度、边缘或一条窄面
- 牌面被手、桌边、筹码、字幕特效等严重遮挡，无法判断具体花色
- 画面太糊、太暗、过曝、运动拖影严重，无法确认具体类别

### 必须明确的边界规则

- 能确认是牌背，就标 `back`
- 不能确认具体万条筒且又不是清晰牌背，就标 `unknown`
- **字牌、花牌统一标 `unknown`**
- 不要为了“尽量凑满 29 类”去猜类别；拿不准时，`unknown` 比猜错更好

## 5. X-AnyLabeling 推荐操作流

推荐“两遍法”：

1. 第一遍只管把所有牌框出来，不纠结类别，先保证框全
2. 第二遍统一回头改类，重点处理易混淆牌和 `back` / `unknown`

推荐操作习惯：

- 先连续画框，保持节奏，不要每画一个框就停下来纠结类别
- 第二遍再批量改类，效率更高，也更容易保持一致性
- 对同一画面里明显相同的牌，优先成组检查，避免一张图里同类牌被标成不同类别
- 对模糊牌先做保守判断，不要为了凑类别硬猜
- 同一视频里如果某种牌型大量出现，先保持一致性，再做全局复查

如果你已经熟悉 X-AnyLabeling，可按自己的习惯配置常用快捷键；核心不是必须使用某个具体按键，而是遵循“先框全、再改类”的两遍工作流。
